How to use adb to grant permissions to my "SetEdit - LoveDoLove" application.

Tutorial Below

1) Click this button download my tools first

2) Unzip the files > Click the install.bat > Then insert 'Y' and click Enter.

3) After install the adb, open your C drive in pc/laptop u will see an adb folder, open it and follow the pictures

4) After open adb folder u will see this

5) then open cmd in this folder

6) then u will see this cmd panel like this

7) Follow the video to enable use debugging

8) After you see the video and enabled the usb debugging

Enter this command : adb shell pm grant lovedolove.setedit android.permission.WRITE_SECURE_SETTINGS

9) Now try to open SetEdit - LoveDoLove application
